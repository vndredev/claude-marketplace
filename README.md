# vndredev Marketplace

Custom Claude Code plugins for development workflows.

## Available Plugins

| Plugin                                                     | Description                              | Install                                    |
| ---------------------------------------------------------- | ---------------------------------------- | ------------------------------------------ |
| [devkit-plugin](https://github.com/vndredev/devkit-plugin) | Clean Architecture + Dev Workflow Plugin | `claude-plugin add vndredev/devkit-plugin` |

## Usage

Browse plugins in `.claude-plugin/marketplace.json` or install directly from the plugin repositories.
